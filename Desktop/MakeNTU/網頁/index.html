?? 此瀏覽器不支援 Web Serial API，請使用 Chrome / Edge / Opera 開啟，或改用「手動測試」欄位輸入。

  1. 點擊「新增區間」，輸入下限與上限（單位：Ω），按「儲存」。區間不可重疊，且上限需大於下限。
  2. 於「即時庫存」中，每個卡片代表一個阻值區間。使用「? / +」按鈕可快速減少或增加庫存數量（不會低於 0）。
  3. 若需修改區間範圍，按下卡片內的「編輯」；要移除則按「刪除」。
  4. 所有設定與庫存數量會自動儲存在本機瀏覽器（Local Storage）。重新整理或關閉網頁後再次開啟仍會保留。
  5. 自動量測：按「連接裝置」選擇 Pico/ESP32 的 USB 序列埠後，量測到的電阻值會自動歸入對應區間並 +1。

提示：可在表單中為每個區間選擇不同的電阻種類；若未選擇，會以上方「電阻種類」的目前選擇作為預設。
